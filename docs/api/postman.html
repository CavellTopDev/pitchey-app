Setup Instructions

1

Download Files

Download both the collection and environment template files using the buttons above.

2

Import into Postman

Open Postman and import both files:

  • Click "Import" in Postman
  • Drag and drop both JSON files
  • Click "Import" to confirm
3

Configure Environment

Select the "Pitchey API Environment" and update these variables if needed:

Variable Default Value Description
baseUrl Production API URL Change to localhost:8001 for local dev
authToken Auto-set after login JWT token for authentication
debug false Set to 'true' for detailed logging
4

Start Testing

Begin with authentication using one of the demo accounts:

Email: alex.creator@demo.com Password: Demo123

The collection will automatically set your auth token for subsequent requests.
