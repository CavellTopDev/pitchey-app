Testing Overview

This page helps you test the enhanced error boundary functionality. The improvements include:

  • Source Maps: Error stack traces now show actual line numbers instead of minified names
  • Enhanced Error Boundary: Comprehensive error logging with copy-to-clipboard functionality
  • Error Reporting: Automatic error reporting to backend API and console logging
  • Better UX: Detailed error information with development tips
